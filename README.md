# Sales_performance_-SQL-Power_BI-
Analyzed multi-year sales data using SQL and Power BI to track revenue, profit, and growth trends. Built interactive dashboards with DAX measures to identify top products, regions, and sales reps, enabling data-driven business decisions.

# Business Objective
* How are sales, revenue, and profit trending over time?
* Which products, regions, and sales reps drive performance?
* Where are leakages (low profit, declining sales)?
* How can management optimize strategy?
